# 死亡診断書: 確定状態の表示調整

## 目的
同意数が 1/1 の場合でも「同意待ち」表示が残る問題を解消し、死亡確定（CONFIRMED）時は完了表示に統一する。

## 方針
- `claim.status === "CONFIRMED"` を最優先で判定し、現在のアクションと申請ステータス本文の表示を「死亡確定済み」に切り替える。
- `confirmedByMe` による「ほかの相続人の同意を待っています」は、CONFIRMED の場合は表示しない。
- API やデータ構造は変更しない。

## UI
- 現在のアクション: 「死亡確定済みです。」を表示し、アクションボタンは非表示。
- 申請ステータス: 同意状況の行は維持し、補足メッセージを「死亡確定済みです。」に切り替える。

## テスト
- CONFIRMED の入力で「死亡確定済みです。」が表示され、同意待ち文言が出ないことを確認する。
