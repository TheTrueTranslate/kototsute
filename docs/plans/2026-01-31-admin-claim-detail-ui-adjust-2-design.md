# 申請詳細 UI 微調整（被相続人プロフィール・ステータス幅）設計

**目的**
- 申請詳細の「ケース概要」を「被相続人プロフィール（氏名のみ）」に置換。
- ステータス表示の横幅を抑え、全幅で広がらない見た目にする。
- 死亡診断書の正当性確認に必要な導線（ファイル閲覧）を維持。

**変更点**
- 表示名を「被相続人プロフィール」に変更。
- 表示内容は `ownerDisplayName` のみ。
- ステータスブロックに `inline` で収まるスタイル調整（レイアウトクラス追加 or 既存クラス利用）。

**スコープ外**
- プロフィール詳細（生年月日/住所など）は未対応。
